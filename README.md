Terra project from VSI

[![CircleCI](https://circleci.com/gh/VisionSystemsInc/terra.svg?style=shield)](https://circleci.com/gh/VisionSystemsInc/terra)

## Requirements

- Python 3.6 or newer
- pipenv
- docker
- bash
- docker-compose

## Getting started

```
source 'setup.env'
just terra sync
just terra run
```
